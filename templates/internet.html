*Puis cliquer sur Connexion

Pour jouer avec un ami sur le même wifi vous avez nesoin de l'adresse ip locale. Pour jouer sur internet vous avez nesoin de l'adresse ip internet. Pour obtenir votre ip internet ou locale sous windows : Paramètres, Réseau et Internet, Paramètres réseau avancés, Propriétés du matèriel et de la connexion. Vous chercherez les valeurs 'adresse IPv4'. L'adresse locale est souvent sous le nom : Wi-Fi et commence normalement par 192.168. L'adresse internet est souvent sous le nom Ethernet.

Vous devez soit communiquer votre ip à votre adversaire et cliquer sur attente de connexion soit saisir l'ip de votre adversaire et cliquer sur connexion après que votre adversaire ce soit mis en attente.

Un match compte deux parties, les deux adversaires jouant en premier à tour de rôle.
